About

This project is a property app that allows users to search for places to buy or rent.

I built it using Next.js, Chakra UI and Bayut API, which provided property info.

Dependencies used:
- Chakra UI for styling.
- emotion/react for styling.
- emotion/styled for styling.
- axios for API requests.
- framer-motion for animation.
- millify to convert long numbers into more readable ones.
- nprogress for a progress bar.
- react-horizontal-scrolling-menu for scrolling images.
- react-icons
